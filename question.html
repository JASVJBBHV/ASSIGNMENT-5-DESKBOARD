Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. Such as we use getElementById() to select a specific element by its ID in the same way we can select elements through class name by getElementsByClassName(). we can slect multiple elements by tag name by using getElementsByTagName(). we can also use query selector to get element by querySelector() and querySelectorAll() command

Question-2:What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the inner HTML code of an element where we use it on the other hand innerText returns the inner text of that element

Question-3: What is event delegation in the DOM?

we use event delegation in DOM when i want to add one event listener to a parent element instead of adding single event listener to each child of it. like when its child element is clicked the event bubbles up to the parent for that event

Question-4:What is event bubbling in the DOM?

event bubling means that when i click on an element the event first happens to that target element and then moves up to its parent

Question-5:How do you create, add, and remove elements using JavaScript?

to create elements we use document.createElement('element') that creates an element like paragraph or other elements. to add element we use appendChild() which add elements where we want to add it, for example:document.getElementById('ID').appendChild(element). to remove elements we use remove() or removeChild()
